What is CSS?

  1. CSS Stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper, or in other media.
  3. is not neccesary CSS saves a lot of work. It can control the layout of multiple web pages all at once
  4. External Stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline -by using the style style attribute inside HTML elements1.
  • Internal - by using a style element in the head section.
  • External - by using a link element to link to external CSS files.
  • The most comman way to add CSS.is the keeps the styles in external CSS files. However, in this tutorial we will use inline and internal styles. because this is easier to demonstate , and easier for you to try yourself.

When comparing CSS class v/s ID , the difference is that CSS class applies a style to multiple element. Id is also special in that you can use special URL to link directly to an element and it's used by JavaScript.

In CSS , selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, you can a style or set of styles can be applied to the element.

There is a wide range of selectors available . two of the most commanly used are class and ID . Both are used to target elements to which a style should be applied.
